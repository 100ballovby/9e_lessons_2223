У нумерованных списков есть два атрибута - start (отвечает за то, с какого числа начнется нумерация списка), type (отвечает за стиль нумерации)

  2. I - большие римские цифры (I, II, IV)
  3. a - маленькие буквы
  4. А - большие буквы
  5. 1 - цифры (ставить не нужно)
